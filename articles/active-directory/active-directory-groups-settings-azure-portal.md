## <a name="how-do-i-find-and-change-the-settings"></a><span data-ttu-id="9034a-105">어떻게 설정을 찾고 변경하나요?</span><span class="sxs-lookup"><span data-stu-id="9034a-105">How do I find and change the settings?</span></span>
