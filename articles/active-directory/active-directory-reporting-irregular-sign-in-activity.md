# <a name="irregular-sign-in-activity"></a>비정상적인 로그인 작업
비정상적인 로그인은 기계 학습 알고리즘 위치와 장치에는 비정상적인 로그인과 결합 하는 "이동 불가능" 상태의 hello 기준에 의해 식별 된 로그인입니다. 이는 해커가 이 계정을 사용하여 성공적으로 로그인했음을 나타낼 수 있습니다.
더 비정상적인 로그인 이벤트가 10 30 일 이하의 범위 내에서 발생 한 경우 전자 메일 알림 toohello 전역 관리자를 전송 됩니다. 있는지 tooinclude 모니터링할 하십시오 aad-alerts-noreply@mail.windowsazure.com 수신 허용 목록에 있습니다.
