# <a name="windows-10-for-the-enterprise-ways-to-use-devices-for-work"></a>엔터프라이즈를 위한 Windows 10: 작업에 장치를 사용하는 방법
Windows 10은 Azure Active Directory를 활용할 수 있는 기능을 제공합니다(Azure AD). Windows 10 장치를 Azure AD에 연결할 수 있어 사용자는 비즈니스 앱 및 리소스에 대한 액세스 권한을 얻기 위해 Azure AD 계정을 사용하거나 Azure ID를 추가하여 Windows에 로그인할 수 있습니다.
