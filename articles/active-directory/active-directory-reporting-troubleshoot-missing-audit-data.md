# <a name="i-cant-find-some-actions-that-i-performed-in-hello-azure-active-directory-activity-log"></a>Hello Azure Active Directory 작업 로그에서 수행한 일부 작업을 찾을 수 없습니다.


## <a name="symptoms"></a>증상

Hello Azure 포털에서에서 일부 작업을 수행 했으며 hello에서 이러한 작업에 대 한 toosee hello 감사 로그를 예상 `Activity logs > Audit Logs` 블레이드에서 하는데 찾을 수 없습니다.

 ![보고](./media/active-directory-reporting-troubleshoot-missing-audit-data/01.png)
 

## <a name="cause"></a>원인

작업은 hello 활동 감사 로그에 즉시 표시 되지 않습니다. Hello 작업을 수행 하는 hello 시간부터 hello 포털에서 tooan 시간 toosee hello 감사 로그는 15 분에서 아무 곳 이나 걸릴 수 있습니다.

## <a name="resolution"></a>해결 방법

15 분 tooan 시간 동안 기다린 후 hello 작업 hello 로그에 표시 되는지 확인 합니다. 여전히 표시되지 않으면 지원 티켓을 제기해 주세요. 살펴보도록 하겠습니다.
