## <a name="how-to-reset-the-password-for-a-user"></a><span data-ttu-id="7cdcc-106">사용자에 대한 암호를 다시 설정하는 방법</span><span class="sxs-lookup"><span data-stu-id="7cdcc-106">How to reset the password for a user</span></span>
1. <span data-ttu-id="7cdcc-107">디렉터리에 대한 전역 관리자인 계정으로 [Azure 포털](https://portal.azure.com) 에 로그인합니다.</span><span class="sxs-lookup"><span data-stu-id="7cdcc-107">Sign in to the [Azure portal](https://portal.azure.com) with an account that's a global admin for the directory.</span></span>
2. <span data-ttu-id="7cdcc-108">**더 많은 서비스**를 선택하고 텍스트 상자에 **사용자 및 그룹**을 입력한 다음 **Enter**를 선택합니다.</span><span class="sxs-lookup"><span data-stu-id="7cdcc-108">Select **More services**, enter **Users and groups** in the text box, and then select **Enter**.</span></span>

3. <span data-ttu-id="7cdcc-110">**사용자 및 그룹** 블레이드에서 **사용자**를 선택합니다.</span><span class="sxs-lookup"><span data-stu-id="7cdcc-110">On the **Users and groups** blade, select **Users**.</span></span>

4. <span data-ttu-id="7cdcc-112">**사용자 및 그룹 - 사용자** 블레이드의 목록에서 사용자를 선택합니다.</span><span class="sxs-lookup"><span data-stu-id="7cdcc-112">On the **Users and groups - Users** blade, select a user from the list.</span></span>

6. <span data-ttu-id="7cdcc-115">**암호 재설정** 블레이드에서 **암호 재설정**을 선택합니다.</span><span class="sxs-lookup"><span data-stu-id="7cdcc-115">On the **Reset password** blade, select **Reset password**.</span></span>
