# <a name="azure-machine-learning-studio-documentation"></a>Azure Machine Learning Studio 설명서

Azure Machine Learning Studio는 데이터에 대한 예측 분석 솔루션을 빌드, 테스트 및 배포하는 데 사용할 수 있는 공동 끌어서 놓기 도구입니다.
자습서, 비디오 및 예제 모델에서 Studio를 사용하여 기계 학습 모델을 빌드하고 배포하는 방법을 보여 줍니다.
