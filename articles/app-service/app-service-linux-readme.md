# <a name="web-app-on-linux-documentation"></a>Linux의 웹앱 설명서
Linux의 웹앱은 현재 공개 미리 보기에 포함되며 고객은 Linux 플랫폼에서 기본적으로 해당 웹앱을 실행할 수 있습니다. 이를 통해 특정 종류의 응용 프로그램에 대한 응용 프로그램 호환성이 더 개선되며, Linux 플랫폼에 호스트된 기존 웹앱을 Azure App Service의 다른 위치로 보다 쉽게 마이그레이션할 수 있습니다.

고객은 Linux 사용 가능 App Service 계획 내에서만 Linux의 웹앱을 만들 수 있으며 이 App Service 계획은 Linux 기반 웹앱만 호스트할 수 있습니다. 

> Linux의 웹앱에 대한 공개 미리 보기 가격은 일반 가격의 50%이며 이 가격은 App Service 계획에서 사용되는 계산 시간을 반으로 나누어 계산합니다.
> 
> 공개 미리 보기 기간 동안 SLA는 제공되지 않습니다.
