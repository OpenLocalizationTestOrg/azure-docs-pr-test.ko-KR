# <a name="azure-data-factory-developer-reference"></a>Azure 데이터 팩터리 개발자 참조
Azure 포털, Azure PowerShell, .NET 클래스 라이브러리 또는 REST API를 사용하여 팩터리를 만들고 모니터링하고 관리할 수 있습니다.

| 메서드 | 리소스 위치 | 개발자 참조 |
| --- | --- | --- |
| Azure 포털 |[https://portal.azure.com/](https://portal.azure.com) |[Azure Data Factory 시작(Azure 포털)](data-factory-build-your-first-pipeline-using-editor.md) |
| Azure PowerShell |최신 [Azure PowerShell](http://go.microsoft.com/?linkid=9811175&clcid=0x409) |[Cmdlet 참조](https://msdn.microsoft.com/library/dn820234.aspx) |
| .NET 클래스 라이브러리 |Azure Data Factory .NET SDK를 사용하여 Azure 데이터 팩터리를 만들고 모니터링하고 관리할 수 있고 .NET 활동을 사용하여 Data Factory를 확장할 수 있습니다. 시작하려면 [Azure Database 파이프라인에서 사용자 지정 활동 사용](data-factory-use-custom-activities.md) 및 [Database .NET SDK를 사용하여 Azure Data Factory 만들기, 모니터링 및 관리](data-factory-create-data-factories-programmatically.md) 문서를 참조하세요.<br/><br/><b>최신 Nuget 다운로드</b><br/>[https://www.nuget.org/packages/Microsoft.Azure.Management.DataFactories/](https://www.nuget.org/packages/Microsoft.Azure.Management.DataFactories/)에서 최신 Azure Data Factory 관리 라이브러리 Nuget 패키지를 다운로드할 수 있습니다.<br/><br/>**Visual Studio에서 패키지 관리자 콘솔 사용**<br/>Visual Studio의 패키지 관리자 콘솔에서 다음 명령을 실행하여 최신 Azure Data Factory 관리 라이브러리를 가져올 수 있습니다.<br/><br/>Install-Package Microsoft.Azure.Management.DataFactories |[.NET SDK 참조](https://msdn.microsoft.com/library/mt415893.aspx) |
| REST API |데이터 팩터리 REST API를 사용하여 Azure 데이터 팩터리를 만들고 모니터링하고 관리할 수 있습니다. |[REST API 참조](https://msdn.microsoft.com/library/dn906738.aspx) |
