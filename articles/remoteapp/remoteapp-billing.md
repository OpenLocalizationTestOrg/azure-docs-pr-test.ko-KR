# <a name="how-to-stop-being-billed-for-azure-remoteapp"></a>Azure RemoteApp에 대한 비용 청구를 중지하는 방법
> [!IMPORTANT]
> Azure RemoteApp은 2017년 8월 31일에 중단되었습니다. 자세한 내용은 [알림](https://go.microsoft.com/fwlink/?linkid=821148) 을 읽어보세요.
> 
> 

할당된 사용자가 없더라도 Azure RemoteApp 컬렉션이 있으면 RemoteApp에 대한 비용이 청구됩니다. 이 청구를 중지하려면 컬렉션을 모두 삭제합니다. 

컬렉션을 삭제하려면

1. Azure 클래식 포털에서 RemoteApp 페이지를 엽니다.
2. 삭제하려는 컬렉션을 선택한 다음 페이지 맨 아래에서 **삭제** 를 클릭합니다.
3. 구독에서 각 컬렉션에 대해 이 단계를 반복합니다. 

각 컬렉션을 삭제하는 데 몇 분 정도 소요될 수 있습니다.
