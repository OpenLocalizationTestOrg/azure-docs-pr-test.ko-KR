# <a name="subscribe-to-an-offer"></a>제품 구독

*적용 대상: Azure 스택 통합 시스템과 Azure 스택 개발 키트*

이제 [제공 하는 서비스를 만든](azure-stack-create-offer.md), 사용자가 구독을 만들 수 있는지 테스트 합니다.

1. [로그인](azure-stack-connect-azure-stack.md) Azure 스택 사용자 포털 (https://portal.local.azurestack.external)를 클릭 하 여 **구독**합니다.

   ![](media/azure-stack-subscribe-plan-provision-vm/image01.png)
2. 에 **표시 이름** 필드, 구독에 대 한 이름을 입력 하 고을 클릭 **제공**를에 제공 하는 중 하나를 클릭는 **제안을 선택** 블레이드에서 하 고 클릭 한 다음  **만들**합니다.

   ![](media/azure-stack-subscribe-plan-provision-vm/image02.png)
3. 만든 구독을 보려면 클릭 **더 많은 서비스**, 클릭 **구독**, 새 구독을 클릭 합니다.  

제공 하는 서비스를 구독 한 후 새 구독의 일부인 서비스를 확인 하기 위해 포털을 새로 고칩니다.

## <a name="subscribe-to-an-add-on-plan"></a>추가 기능 계획을 구독
제안을 추가 기능이 계획에 있는 경우 사용자가 언제 든 지 구독에 추가할 수 있습니다.  

1. 사용자 포털에서 선택 **더 많은 서비스** > **구독**합니다.

2. 해당 구독 클릭 > **계획 추가** 단추를 선택한 추가 기능 계획을 선택 합니다.
