# <a name="web-apps-documentation"></a>Web Apps 설명서

Azure Web Apps를 사용하면 인프라를 관리할 필요 없이 선택한 프로그래밍 언어로 웹 응용 프로그램을 빌드하고 호스팅할 수 있습니다. Azure Web Apps는 자동 크기 조정 및 고가용성을 제공하고, Windows 및 Linux를 지원하며, GitHub, Visual Studio Team Services 또는 Git 리포지토리에서 자동화된 배포를 지원합니다. 빠른 시작, 자습서 및 샘플을 통해 Azure Web Apps를 사용하는 방법을 알아보세요.

Web Apps를 사용하여 클라우드에 첫 번째 응용 프로그램을 배포하는 방법을 알아봅니다.

- [.NET](/azure/app-service/app-service-web-get-started-dotnet)

- [Python](/azure/app-service/app-service-web-get-started-python)
- [HTML](/azure/app-service/app-service-web-get-started-html)
