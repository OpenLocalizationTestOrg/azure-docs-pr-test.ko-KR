## <a name="prerequisites"></a>필수 조건

이 자습서를 완료하려면 활성 Azure 구독이 필요합니다.

> [!NOTE]
> 계정이 없는 경우 몇 분 만에 무료 평가판 계정을 만들 수 있습니다. 자세한 내용은 [Azure 무료 평가판][lnk-free-trial]을 참조하세요.

### <a name="required-software"></a>필수 소프트웨어

Intel NUC의 명령줄에 원격으로 액세스할 수 있도록 데스크톱 컴퓨터에 SSH 클라이언트가 필요합니다.

- Windows에서는 SSH 클라이언트를 포함하지 않습니다. [PuTTY](http://www.putty.org/)를 사용하는 것이 좋습니다.
- 대부분의 Linux 배포판 및 Mac OS는 명령줄 SSH 유틸리티를 포함합니다.

### <a name="required-hardware"></a>필수 하드웨어

Intel NUC의 명령줄에 원격으로 연결할 수 있는 데스크톱 컴퓨터이며

[IoT Commercial Gateway Kit][lnk-starter-kits]. 이 자습서에서는 키트에서 다음 항목을 사용합니다.

- 4G 메모리 및 Bluetooth 확장 카드를 포함한 Intel® NUC Kit DE3815TYKE
- 전원 어댑터

[lnk-starter-kits]: https://azure.microsoft.com/develop/iot/starter-kits/
[lnk-free-trial]: http://azure.microsoft.com/pricing/free-trial/
