
#### <a name="tooconfigure-remote-management-on-cloud-appliance"></a>클라우드 어플라이언스에 tooconfigure 원격 관리

1. StorSimple 장치 관리자 서비스에서 **장치**를 클릭합니다. 선택한 장치 toohello 연결 된 서비스의 클라우드 어플라이언스 hello 목록에서를 클릭 합니다.
    ![StorSimple 클라우드 어플라이언스 선택](./media/storsimple-8000-configure-remote-management-http-device/sca-remote-manage1.png)

2. 너무 이동**설정 > 보안** tooopen hello **보안 설정** 블레이드입니다.

     ![StorSimple 보안 설정](./media/storsimple-8000-configure-remote-management-http-device/sca-remote-manage2.png)

3. Toohello 이동 **원격 관리** 섹션. **원격 관리** 상자를 클릭합니다.
     ![StorSimple 원격 관리](./media/storsimple-8000-configure-remote-management-http-device/sca-remote-manage3.png)

4. Hello에 **원격 관리** 블레이드:

    1. **원격 관리를 사용**이 활성화되어 있는지 확인합니다.
    2. HTTPS를 통해 한 tooconnect hello 기본값인을 참조 하십시오입니다. HTTP를 사용 하 여 tooconnect을 선택할 수 있습니다. HTTP를 통한 연결은 신뢰할 수 있는 네트워크에서만 허용됩니다. HTTP가 활성화되어 있는지 확인합니다.
    3. Hello 블레이드 위쪽에 hello 명령 모음에서 **중... 더 많은** 클릭 하 고 **다운로드 인증서** toodownload 원격 관리 인증서. 위치 어떤 toosave에이 파일을 지정할 수 있습니다. 이 인증서 tooconnect toohello 클라우드 장비를 사용 하는 hello 클라이언트 또는 호스트 컴퓨터에 설치 되어야 합니다.

        ![원격 관리 블레이드](./media/storsimple-8000-configure-remote-management-http-device/sca-remote-manage4.png)
5. 클릭 **저장** 대화 상자가 나타나면 hello 변경 내용을 확인 합니다.
