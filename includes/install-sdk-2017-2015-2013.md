## <a name="setupdevenv"></a>개발 환경 설정
시작하려면 Visual Studio와 Azure SDK에서 개발 환경을 설정하세요.

* Visual Studio 2017에는 Azure SDK가 포함됩니다. VS2017을 사용하는 경우 개발 환경을 위한 추가 설정이 필요하지 않습니다.
* Visual Studio 2015의 경우 다음 링크를 클릭하여 [Visual Studio 2015용 Azure SDK](http://go.microsoft.com/fwlink/?linkid=518003)를 설치하세요.
* Visual Studio 2013의 경우 다음 링크를 클릭하여 [Visual Studio 2013용 Azure SDK](http://go.microsoft.com/fwlink/?LinkID=324322)를 설치하세요.

* Visual Studio가 설치되어 있지 않으면 다음을 사용하여 Azure SDK와 [Visual Studio 2017](https://www.visualstudio.com/)을 설치하세요.

> [!NOTE]
> 사용자 컴퓨터에 있는 SDK 종속성 수에 따라 SDK를 설치하는 시간이 몇 분에서 30분 또는 그 이상이 될 수 있습니다.
