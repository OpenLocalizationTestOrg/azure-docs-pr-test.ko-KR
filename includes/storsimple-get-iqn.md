#### <a name="tooget-hello-iqn-of-a-windows-host"></a>tooget hello Windows 호스트의 IQN
1. Windows 호스트에 hello Microsoft iSCSI 초기자를 시작 합니다. **시작** > **관리 도구** > **iSCSI 초기자**를 클릭합니다.
2. Hello에 **iSCSI 초기자 속성** 창의 hello에 **구성** 탭 선택 하 고 hello에서 hello 문자열을 복사 **초기자 이름** 필드입니다.
   
    ![iSCSI 초기자 속성](./media/storsimple-get-iqn/HCS_iSCSIInitiatorPropertiesFigureIQN-include.png)
3. 이 문자열을 저장합니다.
