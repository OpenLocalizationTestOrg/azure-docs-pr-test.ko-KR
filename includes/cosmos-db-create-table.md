2. Hello 데이터 탐색기 블레이드의 클릭 **새 테이블**, 다음 정보는 hello를 사용 하 여 hello 페이지를 채웁니다.

    테이블 ID|sample-table|새 테이블에 대 한 hello ID입니다. 테이블 이름에는 hello 요구 사항 데이터베이스 id로 문자 동일 합니다. 데이터베이스 이름은 1~255자 사이여야 하며 `/ \ # ?` 또는 후행 공백을 포함할 수 없습니다.
