### <a name="prerequisites"></a>필수 조건
* [Office 365 비디오](https://support.office.com/article/Meet-Office-365-Video-ca1cc1a9-a615-46e1-b6a3-40dbd99939a6) 계정  

논리 앱에서 Office 365 비디오 계정을 사용하려면 먼저 Office 365 비디오 계정에 연결하도록 논리 앱에 권한을 부여해야 합니다. 다행히 Azure 포털의 논리 앱 내에서 이를 쉽게 수행할 수 있습니다.  

Office 365 비디오 계정에 연결하도록 논리 앱에 권한을 부여하는 단계는 다음과 같습니다.  

1. 논리 앱 디자이너에서 Office 365 비디오에 대한 연결을 만들려면 드롭다운 목록에서 **Microsoft 관리 API 표시**를 선택한 다음 검색 상자에 *Office 365 비디오*를 입력합니다. 사용할 트리거 또는 동작을 선택합니다.  
   ![Office 365 비디오 연결 만들기 단계](./media/connectors-create-api-office365video/office365video-1.png)  
2. 이전에 Office 365 비디오에 대한 연결을 만들지 않은 경우 Office 365 비디오 자격 증명을 제공하라는 메시지가 표시됩니다. 이러한 자격 증명을 사용하여 Office 365 비디오 계정의 데이터에 연결하도록 논리 앱에 권한을 부여하고 해당 데이터에 액세스할 수 있습니다.  
   ![Office 365 비디오 연결 만들기 단계](./media/connectors-create-api-office365video/office365video-2.png)  
3. Office 365 비디오에 연결할 자격 증명을 제공 합니다.  
   ![Office 365 비디오 연결 만들기 단계](./media/connectors-create-api-office365video/office365video-3.png)  
4. 연결이 만들어졌으므로 이제 논리 앱의 다른 단계를 진행할 수 있습니다.  
   ![Office 365 비디오 연결 만들기 단계](./media/connectors-create-api-office365video/office365video-4.png)
